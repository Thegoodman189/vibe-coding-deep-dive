# User Stories - PlantsVsMiku's

## Story 1: Start a New Game
**Prioriteit:** Must

Als **speler** wil ik een nieuw spel kunnen starten zodat ik een level kan spelen.

**Acceptatiecriteria:**
- Er is een duidelijke "Start Game" of "Play" knop zichtbaar op het startscherm
- Na klikken wordt level 1 geladen
- De speelwereld/arena is volledig zichtbaar en speelbaar
- Alle game elementen (plants, Miku's, resources) zijn aanwezig

---

## Story 2: Select and Place Characters
**Prioriteit:** Must

Als **speler** wil ik mijn plants op het speelveld kunnen plaatsen zodat ik mijn strategie kan uitvoeren.

**Acceptatiecriteria:**
- Ik kan beschikbare characters selecteren uit een roster
- Ik kan characters op verschillende posities op het grid plaatsen
- Placering is visueel duidelijk weergegeven
- Illegale plaatsingen (buiten grid, op bezette posities) worden voorkomen
- Ik kan characters deselecteren en opnieuw plaatsen vóór de start van battle

---

## Story 3: Manage Resources During Battle
**Prioritiet:** Must

Als **speler** wil ik mijn resources (energie/zon) beheren zodat ik strategische keuzes kan maken.

**Acceptatiecriteria:**
- Resources worden duidelijk in de HUD weergegeven
- Ik kan zien hoeveel resources ik heb en hoeveel ik kan verdienen
- Character-acties kosten resources
- Resources groeien/regeneren over tijd
- Wanneer resources onvoldoende zijn, kan ik bepaalde acties niet uitvoeren (visueel feedback)

---

## Story 4: Win a Level
**Prioriteit:** Must

Als **speler** wil ik een level kunnen winnen zodat ik naar het volgende level kan gaan.

**Acceptatiecriteria:**
- De win-voorwaarde is duidelijk (bijv. alle vijanden verslaan)
- Na het winnen verschijnt een "Level Complete" scherm
- Ik krijg feedback (score, sterren, behaalde objectives)
- Mijn voortgang wordt bijgehouden

---

## Story 5: Understand Game Mechanics with Tutorial
**Prioriteit:** Nice

Als **nieuwe speler** wil ik begrijpen hoe het spel werkt zodat ik niet verloren ben bij mijn eerste game.

**Acceptatiecriteria:**
- Er is een tutorial of help-sectie beschikbaar
- De basisregels worden uitgelegd (placement, resources, combat)
- Character-abilities worden duidelijk beschreven
- Hints of tooltips verschijnen in-game wanneer relevant
- Tutorial kan overgeslagen worden door ervaren spelers

---

## Story 6: Manage Sound and Music Settings
**Prioriteit:** Nice

Als **speler** wil ik muziek en geluidseffecten kunnen in/uitschakelen zodat ik kan genieten van het spel naar mijn voorkeur.

**Acceptatiecriteria:**
- Er is een Settings/Options menu toegankelijk vanuit het startscherm
- Aparte volume-sliders voor muziek en sound effects
- Mute-knoppen voor snelle controle
- Instellingen blijven onthouden bij volgende sessie
- Geluid en muziek reageren correct op instellingen in-game

---

## Story 7: View Level Progress
**Prioriteit:** Nice

Als **speler** wil ik alle 5 beschikbare levels zien met hun huidige status zodat ik weet wat ik al voltooid heb.

**Acceptatiecriteria:**
- Level select scherm toont alle 5 levels
- Voltooide levels zijn duidelijk gemarkeerd (bijv. vinkje, groen)
- Gesloten levels zijn duidelijk als niet-speelbaar gemarkeerd
- Huidige level wordt gemarkeerd
- Level-moeilijkheid/nummer is zichtbaar
- Ik kan vorige levels herspelen

---

## Story 8: Use Power-Ups Strategically
**Prioriteit:** Nice

Als **ervaren speler** wil ik power-ups gebruiken in het spel zodat ik mijn strategie kan verdiepen en moeilijke levels kan winnen.

**Acceptatiecriteria:**
- Power-ups verschijnen op het speelveld of kunnen gekocht worden
- Power-ups hebben duidelijke visuele weergaving
- Effecten van power-ups zijn merkbaar (meer damage, snelheid, healing)
- Power-ups hebben cooldowns of beperkte beschikbaarheid
- Ik kan zien welke power-ups beschikbaar zijn
