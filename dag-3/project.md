## In Scope

### Core Gameplay
- Grid-based battle arena with Plants vs. Miku characters
- Turn-based or real-time strategy gameplay mechanics
- 5 playable demo levels with increasing difficulty
- Character placement and selection system

### Features
- 2-3 plant types with unique abilities
- 2-3 Miku variants with special attacks
- Basic power-ups (energy, health boost, speed)
- Simple item/upgrade system for demo purposes
- Resource management (sun/energy as currency)
- Sound effects and basic music integration

### UI/UX
- Clean, intuitive level select screen
- In-game HUD with player stats and resources
- Game over/win screens
- Settings menu (music/sound volume control)
- Tutorial/help section for new players

### Technical
- Browser-based implementation
- Responsive grid-based rendering

- Save game state for demo progression
